Built With

Eduyomi is a 100% client-side, zero-dependency web application. No frameworks, no build tools, no server. Just HTML, CSS, and vanilla JavaScript — keeping it fast, lightweight, and fully open.

Video playback is powered by the YouTube IFrame API. Content is loaded from user-hosted JSON files on GitHub — meaning anyone can create and share their own curriculum.
